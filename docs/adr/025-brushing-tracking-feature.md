# ADR-025: ブラッシング（歯磨き）記録機能の実装

日付: 2026-05-01
状態: 提案

## 背景

GitHub Issue #2 において、「ブラッシング」という項目が挙げられている。
現在、ユーザーの日常的なタスク（チェックリスト）や家計管理などをPortalアプリで管理しているが、健康管理・習慣化の一環として「ブラッシング（歯磨き）」の記録を管理・追跡する機能が求められている。

## 決定

**Portalアプリ内にブラッシング記録専用のモジュール、または既存のチェックリスト・習慣トラッカーの一部としてブラッシング記録機能を実装する。**

### アプローチの選択肢
1. **既存チェックリストへの統合**
   - 既存のチェックリスト（`pages/checklist.tsx`）の項目として「ブラッシング」を追加し、毎日のタスクとして消化する。
2. **専用の習慣トラッカーの実装**
   - 歯磨きの回数や時間、使用したアイテムなどを詳細に記録できる専用UIを設ける。

初期段階としては、実装コストが低く運用が容易な **既存チェックリストへの統合（選択肢1）** を推奨する。必要に応じて将来的に専用の記録UIへ拡張する。

## トレードオフ

- **メリット:** 既存のインフラ（チェックリストのVault同期や日記への反映など）をそのまま再利用できるため、開発コストが最小限で済む。
- **デメリット:** 詳細な記録（例: 磨いた時間、出血の有無など）を残したい場合には、単なるチェックボックスでは表現力が不足する。

## 次のステップ

- チェックリストのデフォルトデータ、またはユーザーのタスク定義に「ブラッシング」を追加する。
- ユーザーに詳細なトラッキング（時間やメモなど）が必要かヒアリングし、必要であれば拡張モジュールの設計を行う。
